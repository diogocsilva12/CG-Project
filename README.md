# CG-Project --- Computer Graphics with OpenGL

Final project developed for the **Computer Graphics (CG)** course.

**Final Grade:** 18/20

This project focuses on the **generation, rendering, and drawing of 3D
objects using OpenGL**, demonstrating core concepts of real‑time
graphics programming and interactive visualization.

------------------------------------------------------------------------

# Overview

The goal of this project was to implement a graphics application capable
of generating and rendering 3D scenes using **OpenGL**.

The implementation explores several fundamental computer graphics
concepts:

-   3D object generation
-   geometric transformations
-   rendering pipelines
-   scene visualization
-   real‑time interaction

This project demonstrates practical experience with **graphics
programming, rendering pipelines, and scene construction using OpenGL**.

------------------------------------------------------------------------

# Features

The project includes:

-   generation of 3D geometric objects
-   rendering using OpenGL primitives
-   scene transformations (translation, rotation, scaling)
-   camera and view control
-   real‑time visualization

# How to Build and Run

## 1 Clone the repository

``` bash
git clone https://github.com/diogocsilva12/CG-Project.git
cd CG-Project
```

## 2 Compile the project

Example using g++:

``` bash
g++ main.cpp -lGL -lGLU -lglut -o cg-project
```

## 3 Run the program

``` bash
./cg-project
```
